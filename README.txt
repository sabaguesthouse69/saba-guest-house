Saba Guest House Website (Fresh Start)

1) Upload index.html to your GitHub repo root.
2) Create a folder named 'images' in the repo.
3) Upload your photos into /images and rename exactly like this:
   hero.jpg
   room-1.jpg
   room-2.jpg
   bath-1.jpg
   bath-2.jpg
   door-1.jpg
   upi-qr.jpg

4) Enable GitHub Pages:
   Repo Settings -> Pages -> Deploy from branch -> main / (root)
